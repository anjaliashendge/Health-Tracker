#  Health Tracker App

A full-stack health monitoring web app built using ASP.NET Core and SQLite.

##  Features
- Track daily:
  - Sleep hours 
  - Water intake 
  - Exercise 
- Dashboard with real-time data
- Goal tracking (sleep & water)
- REST API with Swagger

##  Tech Stack
- Backend: ASP.NET Core Web API
- Database: SQLite (EF Core)
- Frontend: HTML, CSS, Bootstrap

dotnet run
